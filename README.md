# minishell

### About
Recreation of the UNIX shell command line interpreter in C using only the following functions:

* malloc, free
* access
* open, close, read, write
* opendir, readdir, closedir 
* getcwd, chdir
* stat, lstat, fstat
* fork, execve
* wait, waitpid, wait3, wait4 
* signal, kill
* exit

### Installation
make && ./minishell
